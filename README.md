Python language is used for this assignment

To run this program, users can use Linux operating system. Users can directly run this program from Terminal. To run this program, users have to put "find_route.py" and "input1.txt" files together in a certain path like Desktop.

Users have to follow the rules given below to run this program from terminal:

1)open Terminal
2)write "cd Desktop"
3)write "ls"
4)write "python3.5 find_route.py" (use python installed version instead of "python3.5" if needed)

then it will run project and want some input:

5)Give input for "Enter file name(ex: input1.txt) : "
6)Give input for "Enter Source: "
7)Give input for "Enter Destination: "

After giving input, program will show the result.
